# Determine the URL of the XML version of the Swiss Register of Plant Protection Products

Determine the URL of the XML version of the Swiss Register of Plant
Protection Products

## Usage

``` r
srppp_xml_url()
```

## Value

length one character string

## Examples

``` r
srppp_xml_url()
#> [1] "https://www.blv.admin.ch/dam/de/sd-web/He9bAfs8CmFT/daten-pflanzenschutzmittelverzeichnis-de.zip"
```
